

'passwd USERNAME' - change password. If username is not provided, uses root as default.


'useradd OPTIONS USERNAME'
	Options:
		* -m or --create-home
				Creates a home directory in /home/USERNAME
		* -g or --gid GROUP
				specifies initial group
				Can also do -G or --groups for multiple
			-s or --shell SHELL
				default is usualy bash
				Can use /usr/bin/nologin

'usermod OPTIONS USERNAME'
	Options:
		-G or --groups LIST OF GROUPS
			set these as the user's additional groups
			-a flag will cause these to be appended on to current groups

'userdel OPTIONS USERNAME'
	-r or --remove:
		remove the user's home directory


'groups USERNAME' - display groups that USERNAME is in
